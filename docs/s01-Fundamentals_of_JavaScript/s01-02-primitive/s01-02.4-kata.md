# Задачи для раздела **Примитивы**

- [Multiply](https://www.codewars.com/kata/50654ddff44f800200000004) (8 kyu)  
  Простое умножение чисел (проверка типа `number`).
- [String repeat](https://www.codewars.com/kata/57a0e5c372292dd76d000d7e) (8 kyu)

- [Convert boolean values to strings](https://www.codewars.com/kata/53369039d7ab3ac506000467) (8 kyu)

- [Convert a String to a Number!](https://www.codewars.com/kata/544675c6f971f7399a000e79) (8 kyu)  
  Преобразование `string` в `number`.
- [Bin to Decimal](https://www.codewars.com/kata/57a5c31ce298a7e6b7000334) (8 kyu)  
  Работа с разными системами счисления.
